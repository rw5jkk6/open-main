- 文(statement)と式(expression)の違い
  - 文は値を返さない (例) x=5
    - 文はセミコロンで終わる(Goでは省略してあることがある) 
  - 式は値を返す   
    - 変数、関数名  (例) x, add()
    - 数字        (例) 3        
    - 文字列リテラル      (例) "hello"
    - 式を（）で囲んだもの   (例) (1+2)

- switch式
  - switch 簡易文; 式{  
case 式:  
}

- コードの説明
1. 式のみ(数)
2. 式のみ(文字)
3. 式と文
4. caseに式がある
5. 型アサーション
6. 型アサーションと文
